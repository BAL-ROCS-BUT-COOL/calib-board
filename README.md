### 1. Clone the Repository

Clone the `calibBoard` repository from GitHub:

```bash

```


### 2. Navigate to the Project Directory

```bash
cd calibBoard
```

### 3. Install the Package

Finally, install the `calibBoard` package:

```bash
pip install .
```
